Display Items Page

This page will display a list of the planning item titles in a list view. When the item is clicked it will open up the item to view. A delete button will then be available to delete the item. When the delete button is clicked, it will delete the item from the page. The Display Items Page will also have a button to take the user to the Add Item Page.
